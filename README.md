# Grid & Garage - Motorsport Event Hub

## Description
A responsive front-end web application built to connect local car enthusiasts with global motorsport events, specifically focusing on Formula 1 and the local Cars & Coffee scene. 

This project was developed to showcase core web development fundamentals, emphasizing semantic markup, responsive design, and client-side logic. It features a dynamic DOM manipulation engine for event registration and a robust custom CSS variable architecture for theming.

## Features
- **Client-Side Validation:** Custom JavaScript logic ensures clean data entry for the RSVP system, preventing empty submissions or invalid email formats.
- **Dynamic DOM Updates:** Successfully validated forms instantly generate and inject new user nodes into the DOM and trigger an animated success modal.
- **Dynamic Theming:** Implemented a pure CSS-variable (Custom Properties) dark mode toggle, allowing for O(1) theme swapping without overriding individual element styles.
- **Semantic HTML5:** Built with accessibility and SEO in mind using proper semantic tags (`<nav>`, `<header>`, `<section>`).

## Tech Stack
- Vanilla JavaScript (ES6+)
- HTML5
- CSS3 (Flexbox & CSS Variables)
